Setting TraceViewer preferences

To set the preferences, do the following:

  1. Go to Window > Preferences > Carbide Extensions > TraceViewer.

    The General settings are displayed.

  2. (Optional) Modify the timestamp, undecoded trace values and data format options as needed. See General settings.
  3. From the left-hand navigation pane, select further preferences to modify:
    • Advanced (optional): Define which external filter application to use to filter or modify traces. See Advanced settings.
    • Connection (mandatory): Define the connection method used. See Connection settings.

  4. Click OK to exit the dialog and apply the settings.
