Toolbar

The TraceViewer toolbar and menus contain the following buttons:

Button

When to use?

To connect to the target device. Will change to disconnect button after clicked.

To pause the view update. Will change to un-pause button when clicked.

To maintain (add, remove, edit) current filter settings.

To start an external filter application. Will change to Stop external filter application button when clicked.

To open the search dialog that can be used to find interesting traces.

To maintain (add, remove, edit) current trace coloring settings.

To maintain (add, remove, edit) current trace counting settings.

To maintain (add, remove, edit) current variable value tracing settings.

To clear the current view and the trace file.

To activate Open System Trace components or groups.

To write or open a log file.

To maintain (add, remove, edit) current start / stop trigger settings.

To change connection settings.
TraceViewer extensions

There is also an item on the menubar called TerminalKeyboard Shell, which an extension of the TraceViewer tool.
