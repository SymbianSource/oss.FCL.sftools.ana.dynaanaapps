TraceViewer Plugins settings

This preference page can be accessed from: Window > Preferences > Carbide Extensions > TraceViewer > Plugins.

This page is reserved for TraceViewer plugins to put settings to a common place. To get your preference page under this placeholder, define a category "com.nokia.traceviewer.preferences.PluginsPreferences" to your preference page in the plugin.xml file.
