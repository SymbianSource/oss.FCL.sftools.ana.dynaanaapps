Product description

The TraceBuilder extension for Carbide.c++ IDE is a tool for adding Open System Traces (OST) to the source code.

TraceBuilder consists of two tools:

  • TraceCompiler that parses OST Application Programming Interface (API) format from the source files and generates the corresponding decode files.
  • TraceBuilder UI that enables you to add and maintain OST traces.

TraceBuilder can be activated by selecting the Carbide > TraceBuilder menu item, or via Window > Show View > Other... > Carbide Extensions > TraceBuilder.

Main features

  • UI for adding traces to source code.
  • UI for adding parameters to traces.
  • Function instrumenter.

Known issues

  • TraceBuilder does not support the definition of non-source files on the user interface but you can define them manually by adding the needed file elements to the trace.properties property file. For more information, see the Reference > Tips and tricks section in this help.
