Deleting traces

To delete multiple traces at once, do the following:

  1. From the TraceBuilder menu, select Delete Traces.
  2. (Optional) If you have instrumented traces, you can select if you want to delete Traces from instrumentation, or Traces from groups.
  3. From the tree view, select the traces you want to delete.
  4. Click OK.
