Setting TraceBuilder preferences

To set the preferences, do the following:

  1. Go to Window > Preferences > Carbide Extensions > Trace Builder.

    The General settings are displayed.

  2. (Optional) Modify the Printf Parser Support status.

    If Printf parser support is enabled, TraceBuilder will try to find Kern::Printf and RDebug::Printf macros from your source code and offer to convert them into Open OST traces. You can also set own trace macro for TraceBuilder to look for.

  3. Click OK to exit the dialog and apply the settings.
