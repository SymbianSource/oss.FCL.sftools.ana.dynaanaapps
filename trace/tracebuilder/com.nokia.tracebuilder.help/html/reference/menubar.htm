Menubar

The TraceBuilder menubar contains the following options:

Button

When to use?

To add a new trace to the source code. The trace is added to the location where the cursor is at the moment.

To add a new parameter to the currently selected trace.

To add a new Enumeration or a value to an existing Enumeration.

To parse an Enumeration. The title row of the Enumeration must be selected from the source code.

To check trace properties from the currently selected trace.

To delete the currently selected trace.

To instrument Open System Trace (OST) traces using an instrumentation template. This option can be used, for example, to instrument the entry and exit traces to functions.

To delete multiple selected traces at once. This option can also be used to delete a set of previously instrumented traces.
