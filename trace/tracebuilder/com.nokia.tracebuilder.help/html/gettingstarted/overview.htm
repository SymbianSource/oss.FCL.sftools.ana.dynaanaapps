TraceBuilder overview

TraceBuilder is a tool for adding traces in the Carbide.c++ Integrated Development Environment (IDE).

With the tool, you can easily add Open System Traces (OST) to your component. The OST solution includes the following features:

  • Sending traces from the device to tool.
  • Activating traces from the tool.
  • Standardized and custom trace formats.
  • Decoding and analyzing of traces in tools.
  • Recording traces in log files.
