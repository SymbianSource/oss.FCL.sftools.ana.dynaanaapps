Modifying rules

To modify the tracing rules or counters, do any of the following:

  • To edit rule content, select it and click Edit rule.

    The Edit Rule dialog is displayed, where you can change the range of values as needed. To finish, click OK.

  • To copy a rule as the basis for a new rule, right-click it and select Copy Selected Rule from the context menu. The Edit Rule dialog is displayed, showing the values of the original rule filled in. Change the values as needed and click OK.
  • To deactivate a rule from being executed, clear its check box on the main view:

  • To reset the counters on the main view, select either Reset All Counters from the toolbar, or right-click a rule and select Reset Selected Rule's Counters from the context menu.
  • To remove a rule completely, select it and click Delete Selected Rule .
