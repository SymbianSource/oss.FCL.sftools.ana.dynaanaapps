D_EXC file

D_EXC file is a file which contains crash data. One crash contains two different files:

  • .TXT file (mandatory). If you want to import the D_EXC file, you should specify this file in the Crash Analyser Import Wizard.
  • .STK file (optional). If you want to import this file, ensure that you have stored this file in the same directory as the .TXT file.
