Analysis file

An analysis file means an XML file which is created from a MobileCrash or D_EXC file. Analysis files are created with the Crash Analyser Import Wizard, and imported to the Crash Analyser main view.

MobileCrash files and D_EXC files are data in binary format and for that reason difficult to understand as such. For making the files more readable, the Crash Analyser Import Wizard changes the binary format to XML format. The file extension of an analysis file is .crashxml.
