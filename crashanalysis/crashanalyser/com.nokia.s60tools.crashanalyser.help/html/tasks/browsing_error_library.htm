Browsing error library

To browse the error library:

  1. Click the Open Error Library button.
  2. Select if you want to find individual panics (Panic Search tab), panic categories (Category Search tab), or error codes (Error Search tab).
  3. Select the item from the list.

    You can also write the name of a panic, category, or error code in the topmost field of the view. The tool filters the list according to the text you write in the field.

  4. (Optional) Report a missing panic or error.

If you do not find a panic or error code from the library, you can report the missing one by clicking the Report a panic or error link. A draft e-mail opens (To: and Subject: fields are already filled in). Write a description of what is missing.
