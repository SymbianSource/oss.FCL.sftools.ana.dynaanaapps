Exporting crash files

You can export crash files into a .zip file by taking the following steps:

  1. Select one or more crash files from the list.
  2. Right-click the desired file or files, and select Export and then select the preferred export type:
    • to HTML File: (Save single file as HTML). This type is shown if you have selected only one file.
    • to XML File: (Save single file as crashxml). This type is shown if you have selected only one file.
    • to Zip as HTML: (Save selected files to Zip). Zip file contains HTML files.
    • to Zip as XML: (Save selected files to Zip). Zip file contains crashxml files.
    • All formats to Zip: (Zip file contains html, crashxml and original undecoded binary crash file)

    Note! You cannot export emulator panic files.

  3. Specify the location and name for the .zip file, and click Save.
