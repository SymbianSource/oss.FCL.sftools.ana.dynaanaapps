Getting Started

The Crash Analyser Carbide.c++ Extension is a tool for decoding different types of crash files from binary format to readable format. The following file types are supported: MobileCrash and D_EXC.

You can begin analysing the files by activating the Crash Analyser Import Wizard. Select Carbide > Crash Analyser and then take the following steps:

  1. Activate the Import Wizard.
  2. Select a crash file or the folder where you have crash files.
  3. Select files to be analysed.
  4. Define the location of symbol files, map files, and image files.
  5. Define the reporting settings (optional).
  6. Start analysing the files.

When the files have been imported, they will be displayed in the Crash Analyser view where they can be viewed and analysed.

You can launch the Crash Analyser also by selecting Window > Show View > Other > Carbide Extensions > Crash Analyser.
