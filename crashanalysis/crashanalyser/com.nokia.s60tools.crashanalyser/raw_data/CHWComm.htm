CHWComm

Panic of ECUART module raised by the Serial Comms Server (or "CommsServer").

0

Bad descriptor passed by client.

1

cancel() method called without calling the respective API first. I.e. an attempt was made to cancel an operation that was not started.

2

Incorrect usage of an API for the current DTE/DCE role.

3

Internal error caused by the notification count being negative.
