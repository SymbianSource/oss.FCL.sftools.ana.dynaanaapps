AnalyzeTool properties

AnalyzeTool has additional preferences which can be found by selecting Project > Properties> Carbide Extensions > AnalyzeTool.

The AnalyzeTool advanced preferences are described in the following table:

Options Description
Use rom symbol AnalyzeTool can take advantage of phone image's rom symbol file to pinpoint call stack items located in rom area. To use enable usage of rom symbol and select the corresponding rom symbol files. Selected symbol files are defined only for selected project.
