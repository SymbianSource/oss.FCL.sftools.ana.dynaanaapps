Emulator support

Latest AnalyzeTool version supports running tests also in an emulator environment.

Running analysis

You can run the analysis by opening the data file generated by AnalyzeTool. Click the Open and analyze data file... button.

In the external data gathering mode, analysis data is saved to the %tmp%\epocwind.out file.

In the internal data gathering mode, analysis data is saved to the \epoc32\winscw\c\logs\analyzetool\ folder.
