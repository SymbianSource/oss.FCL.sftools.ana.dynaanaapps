Toolbar

The toolbar in the AnalyzeTool view contains the following icons:

Button Description


Toggles between data output modes.
  • Output to trace gets data using tracing utility connection,
  • Output to file system uses device data file, and
  • Ask always asks which mode to use for each build.

Toggles between activating and deactivating project build with AnalyzeTool.

Starts and stops data capture.
Starts subtest.
Stops subtest.



Change memory leak analysis call stack detail level (Known, Topmost, All).
Opens user selected test run data file or memory leak results file. File can be also selected from recently opened files list. After file open automatically generates memory leak analysis based on the selected file.
Saves raw data file or memory leak results.

For more information, see AnalyzeTool files.
Clears AnalyzeTool made changes and deletes unsaved analysis data.
Opens the AnalyzeTool Preferences page.
