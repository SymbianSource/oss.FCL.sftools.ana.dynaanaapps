Prerequisites

For fully functioning, AnalyzeTool requires the following components:

  • AnalyzeTool command line tool (atool.exe).
  • Symbian component of AnalyzeTool extracted in build environment.
  • AnalyzeTool for target hardware (through SIS or ROM build).
  • Tracing utility Carbide.c++ Extension (any connection method to target hardware that can pass information through RDebug).

Note! If you have taken AnalyzeTool into use through the Nokia internal SymSEE Installer, the required components have been automatically installed to your PC.

AnalyzeTool for target hardware

To run analysis in the target hardware, AnalyzeTool needs to be installed. You can either choose to install AnalyzeTool through an installation package, or add AnalyzeTool to a phone image. You can find the needed files from the c:\apps\atool directory, if you have acquired AnalyzeTool through the Nokia internal SymSEE environment. If you want to install AnalyzeTool from a SIS package, unzip the AnalyzeToolPhone.zip which contains the binaries of the release and debug versions for the supported platforms. In case of ROM image inclusion (with the 3.2 release), AnalyzeTool.iby needs to be included. In case of 5.x release, it is automatically included.

Additions to build environment

S60 version 3.2.x, 5.0, 5.0.1, and 5.1 platforms and MCL releases include AnalyzeTool so if you are using one of those you do not need to take any actions. If you are using a platform which does not include AnalyzeTool, you have to add some AnalyzeTool files to the platform to be able to build with AnalyzeTool. If you have installed AnalyzeTool through the Nokia internal SymSEE environment, these files can be found from c:\apps\atool. The file name is S60_environment_changes_xx.zip where xx specifies the S60 platform version number. To install these files, unzip the file to the directory which contains the epoc32 directory. The zip file includes the necessary directory structure.

Note! Errors about missing AnalyzeTool files are displayed in the problems view if your build environment is missing any necessary AnalyzeTool files.

Checking the AnalyzeTool Carbide.c++ Extension installation

If you intend to use External data gathering mode, check that Tracing utility Carbide.c++ Extension is configured correctly.
