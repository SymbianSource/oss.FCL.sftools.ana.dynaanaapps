Setting the Time Interval

  1. Click and drag your mouse across the graph to select a time period

    2. NOTE You may have to slightly move your mouse back and forth at the end of the graph to continue scrolling.

  2. Click the Zoom to Selected Time Interval icon

OR

  1. Select the Start - End icon to open the Select Time Interval dialog
  2. Enter a start and end time in seconds and click OK

    3. The display graph updates to show the selected start and end times.

You can select time in any of the graphs (CPU load, power, memory) or enter values in the Select Time Interval dialog. When clicking and dragging inside the graph, if you start inside the graph and then drag outside the graph, the end time will be set to the maximum sample time. If you start dragging outside the graph and end the drag operation inside the graph, then the end time is set to the maximum sample time. If you start and end dragging from outside the graph, then the start time and end time are set to zero (0); no samples are chosen.

In the Select Time Interval dialog, a message in black stating "Start time and end time maximum is XX.XXX seconds" is displayed when legitimate start and end times are entered, and the OK button is enabled.

If the End time is less than or equal to the Start time, the dialog box will contain a message in red stating that "End time must be greater than start time". The OK button will be disabled. If start time or end time is greater than the maximum sample time, the dialog box contains a message in red that states the maximum time. The OK button is disabled.

If start time or end time is less than zero or contains a bad float number format, the dialog box will contain a message in red stating that "Start time and end time must be positive values".
