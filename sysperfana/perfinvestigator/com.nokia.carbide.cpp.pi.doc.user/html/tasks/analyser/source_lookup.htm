Using Source Lookup

Use Open Source for Function to examine the source of the selected function.

NOTE The project must be indexed for Source Lookup to operate. See the project’s Properties > C/C++ Indexer panel.

Figure 1. Select Source Lookup from the context menu

Viewing the Source of a Function

  • Right-click the function, select Open Source for Function

    • A source lookup is performed on the project. If a single source is found it is displayed in an editor view. If more than one source is found, a list of sources is presented to choose from.
