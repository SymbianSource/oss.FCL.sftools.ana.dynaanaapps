Select Key Map Profile

When a profiler .dat file is imported and converted to a NPI file you may need to assign a key map profile. Select a key map profile that reflects your device either from:

  • A user defined key map profile (listed under workspace preferences if previously defined)
  • A Carbide built-in that is related to an operating system SDK.
  • A Recommended Profile, if listed.

 

Figure 1. Select Key Press Profile

Select the desired key profile and click Next.
