Showing/Hiding Button Events

Use the Show Button Events menu item to show or hide button event information in the Analyzer view. Button event information is displayed when a checkmark is present, or hidden if no checkmark appears on the menu item.

Figure 1. Button event information shown in an Analyzer view

Changing the Button Event Visibility

  • Choose Investigator > Show Button Events

    • The Analyzer updates its data display to include button event information and a checkmark appears next to the menu item. If a checkmark is already present, then the Analyzer view updates to hide button event information and the checkmark is removed.
