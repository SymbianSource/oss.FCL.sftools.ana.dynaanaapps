Setting Thresholds For Threads, Binaries, and Functions

Thresholds can be set to group threads, binaries, and/or functions according to their amount of CPU usage into a pile you do not care about. For example, you can specify all functions using less than 5 percent of the CPU load to be ignored. Then you can focus on other functions in the graph that occupy more than 5 percent of the CPU.

Note: You must select a time interval on the graph to view statistics for an item; such as the percent of CPU usage and number of times sampled.

Figure 1. Set Thresholds Dialog Box

Table 1. Set Thresholds options
Options Description
Average load < Enter a percentage between .10 and 100 percent of the CPU load
Total samples Enter number of samples to use as threshold. All items sampled less than the number specified will be grouped into one item. The Average load will be calculated based on the Total samples threshold and number of total samples for the item: threads, binaries, or functions.
Use thread threshold for binaries and functions also Check this box to apply the thread threshold value to binaries and functions.
