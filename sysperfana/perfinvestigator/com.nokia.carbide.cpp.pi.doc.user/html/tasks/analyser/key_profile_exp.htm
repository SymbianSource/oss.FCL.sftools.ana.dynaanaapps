Export Key Map Profile

Select Export Profile... to open a wizard dialog to export all user defined workspace profiles to a single XML File. The Export function does not export builtin key profiles. By exporting all your profiles to one XML file you can provide a single XML file for a family of devices.

Enter the full path and XML filename to store your key map profile(s) or select the Key Press Profile... button to browse to a folder and select or define a XML file. Click Finish to create the XML file.
