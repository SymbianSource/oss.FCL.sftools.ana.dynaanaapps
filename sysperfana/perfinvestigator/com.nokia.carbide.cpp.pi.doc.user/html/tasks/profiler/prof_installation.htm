Installing the Profiler on the Target Device

The Performance Investigator Profiler consists of self installing Symbian executable files (.sisx) to be installed on the handheld device.The Profiler agent can be downloaded and installed using the Install remote agents tab in the On-device Connections wizard.

NOTE Nokia PC Suite is used by the Install remote agents pane to transfer the selected agent to the device. Nokia PC Suite is a free download from the Nokia website and can also be used to transfer, store, synchronize, and edit Nokia phone files with a compatible PC.

Figure 1 - New Connection window

Installing the Carbide Profiler

  1. Click Help > On-Device Connections... to open the New Connection wizard
  2. Enter a Connection name, and choose aConnection type from the connection types listed
  3. Click Next
  4. Click the Install remote agent tab (Figure 1)
  5. Select the type and version of Carbide Profiler to install
  6. Click Install
  7. Click Finish once installation is complete

Alternately, save the Profiler agent to your PC by selecting a version of the Carbide Profiler and clicking Save. Once stored on your PC, you can connect and transfer the agent to the target device using one of the following transfer methods:

  • Connecting by Bluetooth
  • Connecting by USB
  • Connecting by Serial Interface
