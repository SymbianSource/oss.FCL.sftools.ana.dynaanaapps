Importing Key Map Profile

Key map profiles corresponding to specific target devices can be imported or exported. The profiles can also be selected, created, modified, and imported/exported in the Performance Investigator preference panel located in Window > Preferences > Carbide.c++ > Performance Investigator.

To import your key map profile, perform the following steps.

Importing a Key Map Profile

  1. Select Performance Investigator Key Press Profile

    2. Select File > Import... from Carbide.c++ to display Import dialog box (Figure 1), and select Performance Investigator Key Press Profile and click Next.

    Figure 1. Carbide.c++ Import window

  2. Select Key Profile XML File

    3. Click Key Press Profile... and browse to and select a key map profile (.xml) file for the device(s) being targeted. Click Next to continue.

    Figure 2. Select Profile XML File

  3. View List of Profiles

    4. If the XML file contains mappings for one or more workspace profiles a Profile(s) Overwrite page will appear (Figure 3). Click Finish to import the workspace key map profile(s).

    Figure 3. Existing Key Profiles in Selected XML File
