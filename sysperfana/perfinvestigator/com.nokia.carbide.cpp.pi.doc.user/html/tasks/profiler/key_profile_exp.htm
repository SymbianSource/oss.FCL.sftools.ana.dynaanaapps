Exporting Key Map Profile

Key map profiles corresponding to specific target devices can be imported or exported. The profiles can also be selected, created, modified, and imported/exported in the Performance Investigator preference panel located in Window > Preferences > Carbide.c++ > Performance Investigator.

To export your key map profile, perform the following steps.

Exporting a Key Map Profile

  1. Select Performance Investigator Key Press Profile

    2. Select File > Export... from Carbide.c++ to display Export dialog box (Figure 1), and select Performance Investigator Key Press Profile and click Next.

    Figure 1. Carbide.c++ Export window

  2. Select or Define Key Profile XML File

    3. Enter a path and xml filename to store all workspace key map profiles or click Key Press Profile... and select a key map profile (.xml) file for the device(s) being targeted. Click Finish to export all workspace profiles to the xml file.

    Figure 2. Select or Define Profile XML File
