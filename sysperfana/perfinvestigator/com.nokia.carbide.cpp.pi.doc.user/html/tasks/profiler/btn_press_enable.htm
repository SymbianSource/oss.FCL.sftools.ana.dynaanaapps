Setting the Button Press Capture Option

Use the Button press capture option in the Tracing menu of the Profiler application to set whether button press information is collected during a trace run of an application. When enabled (On), button press events are stored in the profiling data file. If disabled (Off), then no button press events are recorded.

Figure 1. The Button press capture option in the Profiler Tracing menu

Changing the Button Press Capture Option

  1. Open the Profiler application on the target device
  2. In Tracing menu, set the Button Press Capture option on or off as desired
