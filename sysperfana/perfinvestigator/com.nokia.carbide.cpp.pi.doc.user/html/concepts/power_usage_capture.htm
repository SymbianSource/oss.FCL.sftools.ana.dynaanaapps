Power Usage Capture

Use Power Usage Capture to capture and examine a program’s power consumption in milliwatts. You enable Tracing Option Power Usage Capture in the Profiler and start profiling. Then run the program you want to record power consumption on. Once recording has stopped, the data file is moved to the PC and examined within the Power Usage view (Figure 1) of the Analyzer.
