Performance Investigator Analyzer

The Analyzer is graphical software for analyzing software performance measurement results collected from a device running the Profiler. The analyzer consists of a plugin-architecture. This means that the software consists of a static core-component (Analyzer Core), and several dynamic plug-in components. This is depicted in Figure 1.
