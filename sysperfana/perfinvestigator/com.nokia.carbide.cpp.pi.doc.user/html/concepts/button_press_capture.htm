Button Press Capture

It is possible with Performance Investigator to capture button press information in the Profiler and display it in the Analyser. When doing a Button Press Capture trace on the target device, a timestamp and string for each button pressed is stored.

Figure 1. Button Press Capture information displayed in an analyser view.

You enable the Button Press Capture option in the Profiler to gather button event information. The data is stored in the profiler data file and can later be viewed in the Analyzer. You can choose to show or hide the button event information in the analyzer view or add comments to every button press event. By default, button events are shown.
