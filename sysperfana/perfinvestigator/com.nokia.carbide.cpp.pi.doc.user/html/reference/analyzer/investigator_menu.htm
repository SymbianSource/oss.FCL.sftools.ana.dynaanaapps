Investigator Menu

In the main menu, select Investigator to display menu options available for the Analyzer; as shown in Figure 1.

Figure 1 - Investigator menu options

Table 1. Investigator menu items
Menu Options Description
Show Individual Samples Displays selected items as a vertical line on the graph. Items are displayed at the time interval at which they occurred.
Fill in Graph Loads Items in graph will be filled in with solid color representing each item. If Show Individual Samples is selected, this option has no effect.
Change Threshold Limits... Opens the Set Thread, Binary, or Function Thresholds dialog box.
