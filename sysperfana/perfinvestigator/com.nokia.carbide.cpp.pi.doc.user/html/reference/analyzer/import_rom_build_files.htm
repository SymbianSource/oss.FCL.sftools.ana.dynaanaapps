Select Build Files for ROM

Use the Select build files from ROM dialog to select the build files that correspond to the ROM on the device. Start by selecting the SDK, then the ROM OBY file as well as the ROM symbol file.

Figure 1. Select build files from ROM dialog

Table 1. Select build files from ROM — options
Menu Item Description
Installed SDK Shows a list of the currently installed SDKs. Use Add SDK... to open the Add New SDK dialog where you can define the SDK to use. Use Remove SDK to remove the selected SDK from the devices.xml file and the Installed SDK list. The SDK is not removed from from the drive.

ROM OBY File

Type a ROM OBY file name or click ROM OBY File... to locate a ROM OBY file.
ROM Symbol File Type a ROM symbol file name or click ROM Symbol File... to locate a ROM symbol file.
ROFS OBY/Symbol File Pairs

Click Add... to add a ROFS OBY/Symbol File pair. Click Remove... to remove the selected ROFS OBY/Symbol File pair from the list.
