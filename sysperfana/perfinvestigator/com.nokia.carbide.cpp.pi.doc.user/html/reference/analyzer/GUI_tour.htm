Analyzer GUI Controls

The Analyzer provides several user interface items to control the analyzing process.

The icons available on the toolbar are shown in Figure 1.

Figure 1. Analyzer toolbar icons

Table 1. Toolbar icons
Option Icon Description
Select Time Interval
Select the Start-End icon to select the time interval (in seconds) to be displayed in the current load graph being viewed.
Zoom In
Zooms in on the graph view to display more detail.
Zoom Out
In the graph view, zooms out to display less detail of the selected time interval.
Zoom to Selected Time Interval
Locates the graph's selected time interval and displays it in center of graph.
Show Entire Graph
Displays all profiled data in graph.

There are also several icons available on Title bar which you can use to modify what is shown in the different views. See the Table 2 below for descriptions.

Table 2. Title bar icons
Option Icon Description
Help
Opens the help displaying a short description of the selected view, and a list of links to related information. If you want more information than what is shown in the description, click a link to one of the related topics, or use one of the options available at the bottom of the help view.
Minimize graph
Minimizes the graph so that only the title bar for the graph is displayed. The corresponding table is hidden as well.
Restore graph
Restores a minimized graph and the corresponding table so that they are displayed as normal.
Move graph down
Moves the graph down in the main view, so that the order of the displayed graphs is changed.

If the graph cannot be moved up, this button is disabled.
Move graph up
Moves the graph up in the main view, so that the order of the displayed graphs is changed.

If the graph cannot be moved up, this button is disabled.
Maximize graph
Maximizes the graph so that all the other graphs are hidden and only the selected graph is displayed in the main view.

This also affects the corresponding tables: The tables for the other graphs are hidden and only the table for the selected graph is displayed.
Restore graph
Restores a maximized graph and displays all the graphs and tables simultaneously as normal.

You can also hover the pointer over the graph to view information as shown in Figure 3.
