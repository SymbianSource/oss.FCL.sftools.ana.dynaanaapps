Context Menu Options

Right-clicking within the Analyzer graph and tables will display context popup menus that will vary slightly depending on where you initiated the popup menu. The following table lists and defines Analyzer context menu options.

Note Several menu options allow you to save information to a .csv file that can be opened in a text editor or system editor (such as Excel). Beware that the default filename to save information is new_file.csv. Specify a unique filename if you do not want the default file to be overwritten each time you save information.

Table 1. Analyzer context menu options
Menu Option Description
Binary -> Function -> Thread Displays the Binary, Function, and Thread tables. The graph will display the Threads from which the selected binaries were executed. The functions within the binaries that were called are listed in the functions table.
Binary -> Thread Displays the Binary and Thread tables. The graph will display the threads where the selected binaries were executed.
Binary -> Thread -> Function Displays the Binary, Thread, and Function tables. The graph will display the functions that were called in the selected binaries. The threads table lists the threads where the selected binaries were executed.
Binary ->Function Displays the Binary and Function tables. The graph will display the functions that were called from within the selected binaries.
Binary Only Displays the Binary table and all selected binaries in the graph view.
Change Event Information... Right-click on a button event marker in the Thread Load, Binary Load, or Function Load graph and select this menu option to display a dialog to change related event information.
Change Threshold Limits... Displays the Set Thread, Binary, or Function Thresholds dialog box.
Check All Rows All rows will be checked and represented on the graph.
Check Highlighted Rows This option will place a check mark in all highlighted rows. You can use the shift key and mouse button to select and highlight a continuous sequence of rows. Use the control key to select and highlight random rows. Checked rows will be visible on the graph.
Copy Copies the contents of the selected data from the table to the clipboard.
Copy Drilldown Tables Copies the contents of the entire drilldown table to the clipboard including column titles for easy identification when pasted into a spreadsheet for analysis.
Copy Table Copies the contents of the entire table in the current pane to the clipboard including column titles for easy identification when pasted into a spreadsheet for analysis. The table contents do not have to be selected for the table copy to occur.
Dynamically Rescale Based on Selected Threads Right-click in Memory Usage graph and select this option to rescale the graph based on selected threads.
Function Only Displays Functions only.
Function -> Thread Displays Functions and related Threads. The graph will display threads containing the function calls.
Function -> Thread -> Binary Displays Functions and related Threads and Binaries containing the functions called.
Memory Usage Statistics Right-click in Memory Usage graph and select this menu option to open a Memory Usage Statistics dialog box that reports memory use for the selected time interval.
Open Source for Function Opens the source code file containing the selected function in the editor window.
Power Usage Settings... Right-click in the Power Usage graph and select this menu option to display a dialog box that reports the voltage and battery capacity. The values can be modifed as mentioned in Power Usage Settings.
Power Usage Statistics Right-click in the Power Usage graph and select this option to display a dialog that reports detailed power information for the selected time interval.
Recolor Highlighted Binaries... Displays a color swatch for you to choose a different color for a highlighted row. If multiple rows are selected, then multiple color swatch windows will appear. Change or accept current color for each row selected. Selected rows appear grey.
Recolor Highlighted Functions... Displays a color swatch for you to choose a different color for a highlighted row. If multiple rows are selected, then multiple color swatch windows will appear. Change or accept current color for each row selected. Selected rows appear grey.
Recolor Highlighted Threads... Displays a color swatch for you to choose a different color for a highlighted row. If multiple rows are selected, then multiple color swatch windows will appear. Change or accept current color for each row selected. Selected rows appear grey.
Save All Call Samples for Interval... Right-click in one of the Function Call Analysis tables (when Function Calls tab is selected) and choose this option to save all function calls to a .csv file for the selected time interval. Information provided includes the time, caller address, caller function, caller binary, callee address, callee function, and callee binary.
Save All Event Samples for Interval... Right-click on a button event marker in the Thread Load, Binary Load, or Function Load graph and choose this option to save all button events for the selected time interval.
Save All Power Samples for Interval... Right-click in the Power Usage graph and select this option to save all collected power samples for the selected time interval.
Save Data for Selected Function... Select a function in the Select One Function table (when Function Calls tab is selected) and right-click to choose this option. Detailed information is saved to a .csv file for the selected function.
Save Drilldown Tables... Saves the contents of the entire drilldown table (e.g. Thread -> Binary -> Function) to a specified file including column titles for easy identification when opened in a spreadsheet for analysis.
Save Priority Samples for Checked Threads... Saves the priority setting for Threads that are checked in the Thread table. The first entry in the file shows the last value recorded before entering the selected time interval. Subsequent entries show a change in the priority during the interval. Priority settings are detected when a change is made; thus they are not sampled every millisecond.
Save Samples for Checked Binaries... Right-click in the Binary table (when Binaries tab is selected) and choose this option to save samples for binaries that are checked.
Save Samples for Checked Threads... When Threads tab is selected, save samples for Threads that are checked in the Thread table to a .csv file.
Save Memory Samples for Checked Table Entries... This option is enabled when a time interval is selected. Save memory samples that are checked in the Memory Usage table to a .csv file. The first entry in the file shows the last value recorded before entering the selected time interval. Subsequent entries show changes made during the interval. Memory samples are detected when a change is made; thus they are not sampled every millisecond.
Save Table... Saves the contents of the entire table in the current pane to a specified file including column titles for easy identification when opened in a spreadsheet for analysis. The table contents do not have to be selected for the table to be saved.
Select All Selects the entire table in the current pane.
Show Entire Graph Displays all profiled data in graph.
Show highlighted function's call info This context menu option appears in the top and bottom tables of the Function Calls table - Functions calling the selected function and Functions called by the selected function. Select a function in one of these two tables, right-click and select this option to move the selected function into middle table labeled Select One Function. Functions calling and called by the selected function will be displayed in top and bottom tables.
Show Selected Interval Average Power Line This option appears when you right-click in the Power Usage graph. Check this option to display the average power for the selected time interval.
Sort by Binary Path, then Binary Name In the Binaries and Functions table views, this option will alphabetically sort the path of binaries, then alphabetically sort the binary files by name. Binary path and binary file names are listed in the Binary and Path columns (Binaries table), and In Binary and Path of Binary columns (Functions table).
Switch Key Press Profile Opens a dialog that allows you to change the key map profile associated with the current NPI file.
Thread -> Binary Displays the Thread and Binary tables. The graph will display the binaries that have executed in the selected Thread(s); in relation to the selected time interval.
Thread -> Binary -> Function Displays the Thread, Binary, and Function tables. The graph will display the functions that have been called in the selected binaries that exist in the selected threads.
Thread -> Function Displays the Thread and Function tables. The graph will display the functions that have been called in the selected threads; in relation to the selected time interval.
Thread -> Function -> Binary Displays the Thread, Function, and Binary tables. The graph will display the binaries executed in the selected threads. The functions within the binaries that were called are listed in the functions table.
Thread Only Displays the Thread table and all selected threads in the graph view.
Uncheck All Rows All rows will be unchecked and all related items will be removed from the graph.
Uncheck Highlighted Rows Unchecks highlighted rows and removes the represented item(s) from the graph.
Zoom In Zooms in on the graph view to display more detail.
Zoom Out In the graph view, zooms out to display less detail of the selected time interval.
Zoom to Selected Time Interval Locates the graph's selected time interval and displays it in center of graph.
