Performance Investigator View

Performance Investigator View is available in the bottom right hand corner of the Carbide.c++ workbench after Profiler data file import, or you can open it by selecting Carbide > Performance Investigator.

The view is linked to the Carbide Project view, so it lists the same files, and reflects the changes made in the Project view.

In additions to the names of the imported Trace Files, Performance Investigator view includes the following information for each file:

  • Project: Name of the project where the Profiler data file is located
  • SDK: SDK version used during import
  • Import time: Shows the time when the file was imported, used for sorting the data files in chronological order
  • File Size (KB): Size of the Profiler data file
  • Time (mm:ss): Time used for profiling

You can open an item from the list to PI Analyser view by double-clicking on it.

To remove an item from the view, right-click on it and select remove, or click on the icon on the toolbar.

The view also provides shortcuts from the toolbar

  • to the Performance Investigator Import Wizard, and
  • to the Key Press Profile Import Wizard.
