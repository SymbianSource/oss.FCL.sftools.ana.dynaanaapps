Analyzing Table Data

Performance Investigator (PI) files collect and store a wealth of data which is then displayed in several panes for analysis. However, it is sometimes more convenient to transfer the data to a spreadsheet where better numerical functions and additional charting options are available. To accomplish that you must transfer the data in the PI file to the spreadsheet program. You can do this using the clipboard.

Once a profiler data file is open and visible in an active view, the Edit menu and context menus include the following commands:

Note that additional Menu options are available for saving more precise data; depending on the table in which the context menu is activated.

Menu Option Description
Copy Copies the contents of the current selection to the clipboard. The current selection is made by highlighting specific cells in a table or by selecting a time interval on a graph.
Copy Data for Selected Function Copies the function data, as well as the calling and called associated with the selected function, to the clipboard.
Copy Drilldown Table Copies the contents of the entire drilldown, including column titles, to the clipboard.
Copy Table Copies the contents of the active table, including column titles, to the clipboard. The table contents do not have to be selected for a table copy to occur.
Save Data for Selected Function... Saves the function data, as well as the calling and called associated with the selected function, to a file (.csv).
Save Drilldown Table... Saves the contents of the entire drilldown table (e.g. Threads > Binaries > Functions), including column titles, to a file (.csv).
Save Table... Saves the contents of the active table, including column titles, to a file (.csv).
Select All Selects the entire table in the current pane. When followed by Copy, the entire contents of the table minus the column titles are copied to the clipboard.

When copied to the clipboard, the clipboard data is stored in tab delimited format, which can be pasted into other programs for analysis.
