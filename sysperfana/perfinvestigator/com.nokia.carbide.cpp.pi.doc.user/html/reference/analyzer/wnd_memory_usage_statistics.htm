Memory Usage Statistics

Use the Memory Usage Statistics dialog to view the on-device and memory usage. These values are displayed for the selected time interval as follows:

  • Interval selected in the memory usage view
  • The Used, Free, and Total memory on the device
  • The total memory allocated to all chunks, stacks, heaps, and other code in RAM in the selected time interval
