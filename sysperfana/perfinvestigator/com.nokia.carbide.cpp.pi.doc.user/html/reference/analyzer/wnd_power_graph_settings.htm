Power Usage Settings

Use the Power Usage Settings dialog to to change the voltage and battery capacity. The Power Usage Settings dialog enables:

  • Setting the Voltage value using an integer or floating point number that represents volts (V) and rounded to the neareth 100th of a volt
  • Setting the Battery capacity to an integer that represents battery capacity in milliamperes (mAh)

The power usage graph is redrawn and any statistics displayed are updated as needed.
