Power Usage View

Use the Power Usage view to display the power consumption of the profiled software running on a device.

Figure 1. Power Usage graph

The Power Usage view (Figure 1) provides:

  • graph power usage over time.
  • graph or hide a horizontal line of average power usage during the selected time interval
  • display and change the voltage (v)
  • display and change the battery capacity (mAh)
  • display average power usage (mV) over the selected interval
  • display average energy consumption (mWs) over the selected interval

Left-click and drag the pointer across the graph to select a portion of the power consumption graph. As the pointer is dragged, the start and end lines appear on the graph. Once the mouse click is released, the time interval is updated to indicate the changes made.
