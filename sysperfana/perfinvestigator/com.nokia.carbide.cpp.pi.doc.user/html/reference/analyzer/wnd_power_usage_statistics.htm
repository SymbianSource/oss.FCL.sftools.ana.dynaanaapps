Power Usage Statistics

Use the Power Usage Statistics dialog to view the average power, estimated power, estimated battery life, as well as maximum and minimum power requirements, and variance for the selected interval. These values are displayed as follows:

  • Interval selected in the power usage view
  • Total battery current in milliamp (mAh) and voltage (V)
  • Estimated energy is displayed in millijoules (mJ)
  • Estimated battery life in hours (h) and minutes (m)
  • Average power is displayed in milliwatts (mW), to the nearest tenth
  • Minimum to maximum power are displayed in milliwatts (mW), to the nearest tenth
  • Variance is displayed in milliwatts squared (mW*mW), to the nearest tenth
