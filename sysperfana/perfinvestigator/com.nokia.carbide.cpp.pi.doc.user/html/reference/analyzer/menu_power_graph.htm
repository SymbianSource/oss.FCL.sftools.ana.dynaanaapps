Power Graph Menu

The Power Graph menu is available whenever a .npi or .bap data file is opened that contains power usage trace data. Use it to modify how the graph is calculated, view power statistics, and control how the power information is displayed.

Figure 1. Power Graph menu items

Table 1. Power Graph —menu items
Menu Item Description

Power Usage Graph Settings

Select to open the Power Usage Graph Settings dialog and modify the voltage and battery capacity used by the target device.
Power Usage Statistics Select to display the Power Usage Statistics dialog.
Show Selected Interval Average Power Line Select the Selected Interval Average Power Line option to show a horizontal line on the graph corresponding to the average power usage during the selected time interval.
