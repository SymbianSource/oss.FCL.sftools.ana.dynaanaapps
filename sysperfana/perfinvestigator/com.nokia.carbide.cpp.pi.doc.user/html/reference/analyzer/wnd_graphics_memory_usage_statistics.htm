Graphics Memory Usage Statistics

Use the Graphics Memory Usage Statistics dialog to view details on the graphics memory usage. These values are displayed for the selected time interval as follows:

  • Interval selected in the Graphics Memory Usage view graph
  • The Used, Free, and Total graphics memory on the device
  • The total graphics memory allocated to all processes in the selected time interval
