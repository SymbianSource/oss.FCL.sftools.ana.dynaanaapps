PIProfiler Performance Counter Settings

The figures and instructions below demonstrate the Performance Counter sampler settings for PIProfiler.

Figure 1. Performance counter sampler in PI Profiler UI list (S60)

  1. In the Profiler sampler list (Figure 1), select Performance counter sampler -> Sampler settings to open the Performance counter settings dialog (Figure 2).

    Figure 2. Performance counter settings dialog ( two counters setup)

  2. In the Performance counter settings dialog, select Performance counter 1. The below dialog opens (Figure 3).

    Figure 3. Setting the trace point for performance counter #1.

    Type in the trace point and select OK.

  3. Select performance counter 2: the below dialog opens.

    Figure 4. Setting the trace point for performance counter #2

    Type in the trace point and select OK.

  4. Select Back to return to the sampler list.
  5. Enable the sampler by selecting from the list: Performance counter samplers -> Enable.

    Figure 5. Enabling the sampler

    6. You can view the information on the sampler by selecting Performance counter sampler -> Sampler info. See figure 6.

    Figure 6. Performance counter sampler info dialog
