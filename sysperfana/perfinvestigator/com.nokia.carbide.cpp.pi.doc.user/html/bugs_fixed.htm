Performance Investigator Bugs Fixed

This is a comprehensive list of bugs fixed by release:

  • Bugs Fixed in 2.1.0
  • Bugs Fixed in 2.0.0
  • Bugs Fixed in 1.3.1, 1.3.0, 1.2.2, 1.2.1

Bugs Fixed in v2.1.0

  • 8500 - Fixed an issue that when importing symbols where certain symbols contained erroneous address data.

Bugs Fixed in v2.0.0

  • 2332 - Added ability to change the CPU address/thread sampling interval between 1ms and 10000ms (10 seconds).
  • 4033 - The Select Time Interval dialog has been improved to prevent choosing intervals that extend outside the range of profiled data. Choosing time intervals by using click-drag-release in graphs has been similarly improved. An interval partly outside the graph is converted to one that ends at the last CPU load sample time.

Bugs Fixed in v1.3.1

  • 2550 - Made PI importing more robust to prevent attempts to load malformed or incorrectly labeled .DAT files.

Bugs Fixed in v1.3.0

  • 3664 - Use the pointer to select time in any of the graph views (CPU load, power, memory) or enter the selection values in the Select Time Interval dialog
  • 3933 - Its now possible in PI to select all the rows in a table using Ctrl-A/Select All
  • 4033 - CPU Profiling Data selection has been improved to prevent assigning values that extend outside the range of profiled data
  • 4458 - Added support to Importer for ROFS symbols and multiple ROFS images
  • 4488 - PI will no longer issue a "Function not found" error message when the SYMBOL file function is the last function in the DLL
  • 4785 - The Average Power Mode graphics should be much more readable now
  • 5759 - Updated the Carbide Profiler to work with the N93

Bugs Fixed in v1.2.2

  • 4197 - Fixed an issue that could cause the PI Function tab to hang when zooming to the entire selection
  • 4206 - Corrected a syntax reading error in OBY files that incorrectly resolved the ROFS path and prevented analysis of binaries and functions
  • 4333 - Contextual menus now correctly appear as inactive when tables in the Function Calls tab are blank
  • 4335 - Selecting an empty time period now correctly clears all tables in the Function Calls tab
  • 4336 - Closing an error dialog will no longer cause an exception after attempting to open a source file for the selected function

Bugs Fixed in v1.2.1

  • 3942 - Changing a Button Event Mark value from numerical "5" to string "Five" in the Name box and adding a comment should no longer corrupt the NPI data file
  • 3946 - The Memory Statistics dialog now shows a value for the highest memory usage found in the given interval (see 4018)
  • 3951 - The Memory Usage Data view revised to show more meaningful thread/process names and documentation revised to clarify column contents
  • 3959 - Power usage documentation revised to clarify constraint of device timing when doing power measurements
  • 3979 - Analyzer updated to better handle data files with incorrectly structured address/thread traces
  • 4018 - Added the Memory Statistics dialog with max RAM usage information to the documentation (see 3946)
